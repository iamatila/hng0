## Getting Started

### Prerequisites

- Go 1.21 or higher


### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd hng0
```

2. Install dependencies:
```bash
go mod tidy
```

5. Run the application:
```bash
go run main.go
```

The server will start on `http://localhost:8000`
